# BreathingBP Support

Thank you for using BreathingBP.

If you have questions, problems, or feedback, please use one of the options below:

- **Email support:** sjpbailey1961@icloud.com  
- **Bug reports / feature requests:** Open an “Issue” on this GitHub page  
- **App info:** BreathingBP is a wellness and relaxation app.  
  It reads blood pressure and heart rate values that already exist in the Apple Health app
  and helps you track how guided breathing affects your wellness over time.
  It does **not** connect directly to any medical devices.

> **Important:** BreathingBP is not a medical device and does not provide medical advice.
> Always consult your doctor or qualified health professional before making any medical decisions.
